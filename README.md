## ✨ Features

### 🧮 Calculator Modes

| Mode | Description |
|------|-------------|
| **Basic** | Addition, subtraction, multiplication, division, percentages, memory functions |
| **Scientific** | Trigonometry, logarithms, powers, roots, factorials, π, e |
| **Programming** | Binary, octal, decimal, hexadecimal conversions, bitwise operations |
| **Graphing** | Plot 2D equations with smooth animations *(Pro)* |
| **Matrix** | Matrix operations, determinant, inverse *(Pro)* |
| **Statistics** | Mean, median, mode, standard deviation, regression *(Pro)* |
| **Unit Converter** | Length, weight, temperature, area, volume, time |

### 🎨 Design Features

- ✅ Premium glassmorphism UI with backdrop blur effects
- ✅ 6 luxury themes (Midnight, Ocean, Sunset, Forest, Royal, Custom)
- ✅ Dark & light mode with smooth transitions
- ✅ 60fps Framer Motion animations
- ✅ 3D tilt effects & custom cursor
- ✅ Animated gradient mesh backgrounds
- ✅ Fully responsive (mobile, tablet, desktop, ultrawide)
- ✅ WCAG 2.1 AA accessible

### ⚡ Functionality

- 📜 Calculation history (last 100 calculations)
- 📋 One-click copy to clipboard
- 🔗 Share calculations via link
- 💾 Export results as PDF, PNG, CSV
- ⌨️ Full keyboard shortcut support
- 🌐 Multi-language support (10+ languages)
- 📱 PWA — install as app, works offline
- 🔊 Optional sound effects & haptic feedback

Follow these steps to run the project locally:

### Prerequisites

Make sure you have the following installed:

- **Node.js** 18+ ([Download](https://nodejs.org/))
- **npm** 9+ / **yarn** / **pnpm**
- **Git**
