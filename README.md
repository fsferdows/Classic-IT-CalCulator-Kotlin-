## ✨ Features

### 🧮 Calculator Modes
- **Basic** — Addition, subtraction, multiplication, division, percentages
- **Scientific** — Trigonometry, logarithms, powers, roots, factorials
- **Programming** — Binary, octal, decimal, hexadecimal conversions
- **Graphing** *(Pro)* — Plot 2D equations with animations
- **Matrix** *(Pro)* — Add, multiply, inverse, determinant
- **Statistics** *(Pro)* — Mean, median, standard deviation, regression
- **Financial** *(Pro)* — EMI, ROI, compound interest
- **Unit Converter** — Length, weight, temperature, area, volume, time

### 🎨 Design
- Premium glassmorphism UI with smooth Framer Motion animations
- 6 luxury themes (Midnight, Ocean, Sunset, Forest, Royal, Custom)
- Dark & light mode with seamless transitions
- 3D tilt effects, custom cursor, animated backgrounds
- Fully responsive (mobile, tablet, desktop, ultrawide)
- WCAG 2.1 AA accessible

### ⚡ Functionality
- Calculation history (100 free / unlimited pro)
- Cloud sync across devices
- Keyboard shortcuts (full support)
- Copy & share results via link
- Export to PDF, PNG, CSV
- Multi-language support (10+ languages)
- PWA — install as app, works offline
- Sound effects & haptic feedback

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/classicit-calculator.git

# Navigate to project
cd classicit-calculator

# Install dependencies
npm install

# Setup environment variables
cp .env.example .env.local

# Start development server

npm run dev
